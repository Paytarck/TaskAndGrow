How It Works

⭐ Yearly Vision Board

The Yearly page shows all 12 months as cards. Each card displays how many tasks you've planned and completed for that month.

📍 Month-Specific Tasks

Inside each month (this page), you can add tasks tied to a specific day of the month. Tasks are sorted by day so you always know what's coming next.

⏰ Auto Time

If you don't set a reminder time when adding a task, it automatically defaults to 11:30 AM. You can always edit a task to change the time.

✅ Completing Tasks

Tap the circle next to a task to mark it complete. Note: completed yearly tasks are locked — they cannot be unchecked, keeping your history accurate.

🔍 Search & Sort

Use the Search button in the bottom nav to filter tasks by name or description. Sort by day (ascending/descending) or alphabetically using the sort pills.

🔗 Monthly Sync (Data Management)

When you enable Monthly Sync in Data Management, all tasks in a yearly month are automatically pushed to your Monthly view when that calendar month arrives. Completing the task in Monthly will also check it off here.

📱 Navigation

Use the bottom navigation bar to jump between Home, Search, Add Task (+), Refresh, and Settings. From the yearly month detail page the Settings icon leads to this page.

Data Management

Monthly Sync

When enabled, tasks from each yearly month are automatically added to your Monthly view when that calendar month begins. Checking a task in Monthly will also mark it complete here.

Auto Sync

When enabled, yearly tasks are automatically synced to Monthly the moment that calendar month arrives — no manual action needed. Works in the background whenever the app is opened.

✅ Sync Active

Yearly tasks will be synced to monthly automatically.

📋 What happens when sync is ON

• When you visit a yearly month page whose calendar month matches the current month, all tasks from that month are copied to Monthly Tasks automatically.


• Each copied task links back to the original yearly task. If you check it off in Monthly, it gets checked here too — in real time.


• Sync only runs once per month (per year) to avoid duplicates. If you add new tasks later, use the Re-sync Now button below.

Re-sync Current Month

Push all tasks from the current yearly month into Monthly Tasks again. Safe to run — won't create duplicates.

Clear Synced Tasks

Remove all tasks that were synced from yearly into monthly. This does not affect your original yearly tasks.
