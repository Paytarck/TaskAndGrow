Month Management

Jump to Next Month

Automatically import all recurring ("All Months") tasks to the current month and reset their completed status.

Auto-Jump

Automatically advance to the next month when it starts, importing recurring tasks.

How It Works

🎯 Task Scopes

📌 This Month Only: Task appears only in the current month. When you jump to the next month, it's archived.

🔁 All Months: Task is recurring. When you jump to the next month, it's imported as a fresh task (marked incomplete).

🔔 Reminders

Set a reminder time for any task and TaskFlow will notify you at the scheduled time. Browser notifications must be enabled.

🔍 Search & Sort

Use the search button in the navigation to find tasks by name or description. Sort by date, alphabetical order, or completion status.

✅ Completing Tasks

Check off tasks as you complete them. Your progress bar updates in real-time. Completed tasks can be cleared in bulk from any view.

📱 Mobile Friendly

TaskFlow is designed to work seamlessly on all devices. Use the bottom navigation to switch between Daily, Monthly, and Yearly views.

Daily Sync

Daily Sync

When enabled, tasks due today from your Monthly list are automatically copied into your Daily todo. They appear with an "Imported from Monthly" label.

Auto Daily Sync

Automatically syncs today's monthly tasks to Daily every time you open the app — no manual action needed.

✅ Daily Sync Active

Today's monthly tasks will be synced to Daily.

Sync Today's Tasks Now

Immediately push all of today's monthly tasks into your Daily todo. Won't create duplicates.

Clear Synced Daily Tasks

Remove all tasks that were synced from Monthly into Daily. Your original monthly tasks are not affected.

📋 How Daily Sync works

• Only tasks whose due day matches today's date are synced into Daily.


• Synced tasks appear in Daily with a badge: 📥 Imported from Monthly.


• Each sync runs once per day to avoid duplicates. Use Sync Now if you add new tasks later.


• Completing or deleting a synced task in Daily does not affect the original Monthly task.
